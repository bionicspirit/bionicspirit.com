Configure file paths to open in IntelliJ IDEA #

The idea command line utility isn’t compatible with iTerm’s settings, an extra script is needed:

#!/bin/sh

GOTO_FILE="$1"
GOTO_LINE="$2"
IDEA_PATH="$(which idea)"

if [ -z "$IDEA_PATH" ]; then
    IDEA_PATH="/usr/local/bin/idea"
fi

if [ -z "$GOTO_FILE" ]; then
    exec "$IDEA_PATH"
elif [ -z "$GOTO_LINE" ]; then
    exec "$IDEA_PATH" "$GOTO_FILE"
else
    exec "$IDEA_PATH" --line "$GOTO_LINE" "$GOTO_FILE"
fi

Then in iTerm’s settings, goto Profiles -> Advanced -> Semantic History, and set Run command ... to:

$HOME/bin/idea-goto \1 \2
